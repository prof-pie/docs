**Stickee** is a modern sticky notes application built for anyone who thinks visually and wants their ideas organized — developers, students, project managers, writers, and anyone in between. It combines the tactile simplicity of physical sticky notes with local storage, rich customization, and smart productivity features. Stickee runs in your web browser and as a native desktop app on Windows, macOS, and Linux.

## Note Attributes

Every Stickee note is built from a set of attributes you can set at creation time or edit later:

| Attribute | Details |
|---|---|
| **Title** | Optional. Up to 20 characters. Displayed prominently on the note card. |
| **Content** | Required. Up to 1,500 characters per note. Supports automatic URL detection. |
| **Color** | Choose from 10 options: yellow, pink, blue, green, purple, orange, teal, lavender, peach, and mint. |
| **Status** | Categorize your note as **To-Do**, **Doing**, or **Done**. |
| **Pinned** | Pin important notes to keep them at the top of your board at all times. |
| **Archived** | Move completed or low-priority notes out of your main board without deleting them. |
| **Emoji Reactions** | Add emoji reactions to notes for quick visual categorization; reaction counts are tracked per note. |
| **Checklists** | Attach a nested task list to any note, toggle items complete, and add or delete tasks dynamically. |

<Note>

</Note>

## Platforms

Stickee runs wherever you need it:

- **Web browser** — open the Stickee web app in any modern browser. No installation required.
- **Desktop app (Windows, macOS, Linux)** — install the native desktop application built with [Tauri](https://tauri.app) for a fast, lightweight experience that lives in your taskbar or dock.

Use the export/import feature to transfer your notes between devices or create backups.
