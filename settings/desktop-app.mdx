Stickee is available as a native desktop app built with [Tauri](https://tauri.app), giving you a fast, lightweight experience on Windows, macOS, and Linux — no browser tab required.

## Why Use the Desktop App

<CardGroup cols={2}>
  <Card title="Native Window Management" icon="window-restore">
    The desktop app behaves like any other app on your operating system — you can minimize it, pin it to your taskbar or dock, and switch to it instantly.
  </Card>
  <Card title="Full Keyboard Shortcut Support" icon="keyboard">
    All keyboard shortcuts — including `Alt`+`P` for the Quick Note popup — work reliably within the Stickee window without any browser in the way.
  </Card>
  <Card title="Faster Load Times" icon="gauge-high">
    The desktop app loads faster than opening a browser tab. Your notes are ready as soon as the app window appears.
  </Card>
  <Card title="No Browser Required" icon="ban">
    Keep your notes app completely separate from your browsing. The desktop app runs independently — no tabs, no extensions, no distractions.
  </Card>
</CardGroup>

## Downloading the App

Visit the [Stickee website](https://stickee-info.simicodes.xyz) to download the installer for your platform.

<CardGroup cols={3}>
  <Card title="Windows" icon="windows">
    Download the `.exe` or `.msi` installer and run it to install Stickee on Windows.
  </Card>
  <Card title="macOS" icon="apple">
    Download the `.dmg` file, open it, and drag Stickee into your Applications folder.
  </Card>
  <Card title="Linux" icon="linux">
    Download the `.AppImage` for a portable install, or the `.deb` package for Debian-based distributions.
  </Card>
</CardGroup>

## First Launch

<Steps>
  <Step title="Open the app">
    Launch Stickee from your Start menu, Applications folder, or app launcher after installation completes.
  </Step>
  <Step title="Start taking notes">
    The app opens directly to your note board. Click the **+** button to create your first note, or use the export/import feature to bring in existing notes.
  </Step>
</Steps>

## The Web App

If you prefer not to install anything, you can use Stickee directly in your browser. Every feature works in the web app — the desktop app simply adds native OS integration on top.

<Info>
  Use the export/import feature to transfer your notes between the desktop app and web app, or to create backups.
</Info>

| | Desktop App | Web App |
|---|---|---|
| Installation required | Yes | No |
| Works offline | Yes | Yes |
| Works without a browser | ✅ | ❌ |
| Native window management | ✅ | ❌ |
| All core features | ✅ | ✅ |
| Local storage | ✅ | ✅ |

<Note>
  Both the desktop app and web app store notes locally. Use the export/import feature to transfer notes between devices or backup your data.
</Note>

<Tip>
  Install the desktop app on your primary computer for the best experience, and use the web app when you are on a borrowed or secondary device.
</Tip>
