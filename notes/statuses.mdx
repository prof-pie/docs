Every note in Stickee carries a status that shows where it sits in your workflow. Statuses turn your sticky-note board into a lightweight task tracker — no extra configuration required. Move a note from **To-Do** to **Doing** to **Done** as you work.

## Available Statuses

| Status | Badge color | Meaning |
|--------|-------------|---------|
| **To-Do** | 🔴 Red | The task hasn't been started yet. |
| **Doing** | 🔵 Blue | You're currently working on it. |
| **Done** | 🟢 Green | The task is complete. |

## Setting a Status

You can set or change a note's status at any time:

- **When creating a note** — click one of the status badges (**To-Do**, **Doing**, or **Done**) in the Add Note dialog before you save. The default is **To-Do**.
- **When editing a note** — open the note's detail view, select a new status from the status selector, then click **Save Changes**.

## Filtering by Status

Use the **search bar** and type a status name to filter your board to only notes that match. For example:

- Type `doing` to see every note currently in progress.
- Type `done` to review everything you've completed.

The filter is case-insensitive and updates in real time as you type. Clear the search bar to restore the full board.

## Workflow Tips

<Steps>
  <Step title="Start with To-Do">
    Create every new task as **To-Do**. Your board becomes an instant to-do list of everything that needs attention.
  </Step>
  <Step title="Move to Doing when you start">
    Change the status to **Doing** as soon as you begin working on something. A quick glance at the blue badges tells you exactly what's in flight.
  </Step>
  <Step title="Mark Done and clean up">
    Set the status to **Done** when you finish. Then archive or delete completed notes to keep your board focused on what's still active.
  </Step>
</Steps>

<Tip>
  Combine **status** and **color** for powerful visual organization. For example, use **blue** notes for work tasks and set the status to **To-Do** (red badge) for urgent ones — the color-and-badge combination makes priorities obvious at a glance.
</Tip>
