Stickee's **search bar** filters your entire note board in real time — the moment you start typing, matching notes surface instantly and non-matching ones fade away. No need to press Enter or navigate to a separate search page.

## How to Search

<Steps>
  <Step title="Click the search bar">
    The search bar sits at the **top of your board** with a magnifying-glass icon on the left. Click it to start typing. (The search bar is only available on desktop — it is hidden on smaller screens.)
  </Step>
  <Step title="Type your query">
    Enter any word, phrase, color name, or status. Stickee filters your notes **instantly as you type** — results update with every keystroke.
  </Step>
  <Step title="Browse the filtered results">
    Only notes that match your query remain visible. Scroll through them as normal.
  </Step>
  <Step title="Clear the search">
    Delete the text in the search bar to remove the filter and return to your full board.
  </Step>
</Steps>

## What You Can Search

Stickee's search covers every visible attribute of your notes:

<CardGroup cols={2}>
  <Card title="Note content" icon="file-lines">
    Search for any word or phrase inside a note's body text.
  </Card>
  <Card title="Note titles" icon="heading">
    Search by the title you've given a note.
  </Card>
  <Card title="Colors" icon="palette">
    Type a color name — **blue**, **yellow**, **pink**, **green**, **purple**, and more — to show all notes of that color.
  </Card>
  <Card title="Statuses" icon="circle-check">
    Type **to-do**, **doing**, **done**, or **backlog** to filter notes by their current status.
  </Card>
</CardGroup>

## Search Tips

<AccordionGroup>
  <Accordion title="Search is case-insensitive">
    You can type in any combination of upper and lower case — **MEETING**, **Meeting**, and **meeting** all return the same results.
  </Accordion>
  <Accordion title="Combine keywords to narrow results">
    Type multiple words to find notes that match all of them. For example, typing **"meeting done"** surfaces notes that contain the word "meeting" *and* carry the **Done** status — great for reviewing completed meeting notes.
  </Accordion>
  <Accordion title="Search by color to group visually">
    If you use note colors as a category system (e.g., yellow for personal, blue for work), type the color name to instantly pull up all notes in that group.
  </Accordion>
</AccordionGroup>

<Tip>
  Searching by status — for example, typing **"doing"** — is the fastest way to see everything currently in progress across your entire board without changing your layout or view mode.
</Tip>
